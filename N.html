1

ข้อตกลงทั่วไป

ยินดีต้อนรับสู่ระบบเช็คอินเรียนออนไลน์ ("ระบบ") การเข้าถึงหรือการใช้งานระบบนี้ หมายความว่าคุณยอมรับและตกลงปฏิบัติตามเงื่อนไขการใช้งานที่กำหนดไว้ในเอกสารนี้

สำคัญ

การใช้งานระบบนี้ถือว่าคุณได้อ่านและยอมรับเงื่อนไขทั้งหมดแล้ว

1.1 ขอบเขตการใช้งาน

ระบบนี้ออกแบบมาเพื่อใช้งานภายในสถานศึกษาเท่านั้น สำหรับการเช็คอินเข้าเรียน การจัดการตารางเรียน และการติดตามผลการเรียน

1.2 ผู้ใช้งาน

ระบบนี้มีไว้สำหรับนักเรียน นักศึกษา ครูอาจารย์ และเจ้าหน้าที่ของสถานศึกษาเท่านั้น

2

การใช้งานระบบ

2.1 การสมัครสมาชิก

  • ผู้ใช้จะต้องได้รับสิทธิ์การเข้าใช้งานจากสถานศึกษา
  • ข้อมูลที่ให้ไว้ในการสมัครต้องถูกต้องและเป็นจริง
  • ผู้ใช้มีหน้าที่ปรับปรุงข้อมูลส่วนตัวให้เป็นปัจจุบัน

2.2 การใช้งาน QR Code

  • QR Code ใช้สำหรับการเช็คอินเข้าเรียนเท่านั้น
  • ห้ามแชร์หรือให้ผู้อื่นใช้ QR Code ของคุณ
  • QR Code จะมีอายุการใช้งานจำกัด

ข้อห้าม

ห้ามใช้ระบบในการปลอมแปลงข้อมูลการเข้าเรียน หรือช่วยผู้อื่นเช็คอินแทน

2.3 ความปลอดภัย

  • ผู้ใช้ต้องรักษาความปลอดภัยของบัญชีผู้ใช้งาน
  • ไม่แชร์รหัสผ่านหรือข้อมูลการเข้าใช้งาน
  • แจ้งทันทีหากพบการใช้งานที่ผิดปกติ
3

สิทธิ์และหน้าที่ของผู้ใช้

3.1 สิทธิ์ของผู้ใช้

  • เข้าถึงและใช้งานระบบตามที่ได้รับสิทธิ์
  • ดูข้อมูลการเรียนและการเข้าเรียนของตนเอง
  • ขอความช่วยเหลือจากเจ้าหน้าที่
  • แจ้งปัญหาหรือข้อเสนอแนะ

3.2 หน้าที่ของผู้ใช้

  • ใช้งานระบบอย่างถูกต้องและเหมาะสม
  • ไม่กระทำการที่เป็นอันตรายต่อระบบ
  • ปฏิบัติตามกฎระเบียบของสถานศึกษา
  • รายงานข้อผิดพลาดหรือปัญหาที่พบ

การใช้งานที่ถูกต้อง

ใช้งานระบบตามวัตถุประสงค์เพื่อการศึกษาและการเรียนการสอนเท่านั้น

4

ข้อมูลส่วนบุคคล

4.1 การเก็บรวบรวมข้อมูล

ระบบจะเก็บข้อมูลที่จำเป็นสำหรับการใช้งาน ได้แก่:

  • ข้อมูลส่วนตัวพื้นฐาน (ชื่อ, นามสกุล, รหัสนักเรียน)
  • ข้อมูลการเข้าเรียน และเวลาเช็คอิน
  • ภาพถ่ายจากกล้องสำหรับการเช็คอิน
  • ข้อมูลการใช้งานระบบ

4.2 การใช้ข้อมูล

  • ข้อมูลจะใช้เพื่อการจัดการเรียนการสอนเท่านั้น
  • ไม่นำข้อมูลไปใช้เพื่อการอื่นโดยไม่ได้รับอนุญาต
  • ข้อมูลจะถูกเก็บตามกฎหมายคุ้มครองข้อมูลส่วนบุคคล

4.3 การแชร์ข้อมูล

ข้อมูลจะถูกแชร์ให้กับ:

  • ครูอาจารย์ที่เกี่ยวข้อง
  • เจ้าหน้าที่ของสถานศึกษา
  • ผู้ปกครอง (กรณีนักเรียนเยาวชน)
5

ข้อจำกัดความรับผิดชอบ

5.1 ความพร้อมใช้งาน

ระบบอาจมีช่วงเวลาที่ไม่สามารถใช้งานได้ เนื่องจากการบำรุงรักษาหรือปัญหาทางเทคนิค

5.2 ความถูกต้องของข้อมูล

  • ระบบจะพยายามรักษาความถูกต้องของข้อมูล
  • ผู้ใช้มีหน้าที่ตรวจสอบข้อมูลของตนเอง
  • แจ้งแก้ไขหากพบข้อมูลที่ผิดพลาด

5.3 การใช้งานที่ผิดวัตถุประสงค์

สถานศึกษาไม่รับผิดชอบต่อความเสียหายที่เกิดจากการใช้งานที่ผิดวัตถุประสงค์หรือผิดเงื่อนไข

6

การเปลี่ยนแปลงเงื่อนไข

6.1 สิทธิ์ในการแก้ไข

สถานศึกษาขอสงวนสิทธิ์ในการแก้ไขเงื่อนไขการใช้งานนี้ได้ตลอดเวลา

6.2 การแจ้งเตือน

  • จะมีการแจ้งเตือนผู้ใช้งานล่วงหน้าอย่างน้อย 30 วัน
  • การแจ้งเตือนจะผ่านระบบหรือช่องทางอื่นที่เหมาะสม
  • การใช้งานต่อไปถือว่ายอมรับเงื่อนไขใหม่

การติดตามการเปลี่ยนแปลง

ผู้ใช้ควรตรวจสอบเงื่อนไขการใช้งานเป็นระยะ ๆ
